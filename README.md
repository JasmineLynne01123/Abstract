# Abstract - Responsive Help Center Clone

## 💻 Project Overview
This project is a high-fidelity, **fully responsive** reconstruction of the Abstract Help Center landing page. It was built to demonstrate advanced proficiency in modern **HTML5** and **CSS3**, focusing specifically on creating a seamless user experience across all device types.

## ✨ Key Features
* **Mobile-First Design:** Optimized for smartphones with stacked layouts and touch-friendly elements.
* **Tablet & Desktop Scaling:** Uses advanced CSS techniques to transition smoothly between screen sizes.
* **Navigation & UI:** A clean, functional header and hero search section modeled after the official Abstract interface.
* **Resource Grid:** A balanced, multi-column layout for help categories that adjusts based on screen width.

## 🛠️ Technologies Used
* **HTML5:** Semantic markup for structure and accessibility.
* **CSS3:** Custom styling, Flexbox layouts, and Media Queries for responsiveness.
